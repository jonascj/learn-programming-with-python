########################
Processing (Python Mode)
########################

.. image:: figs/processing-web.png
    :align: center
    :target: https://py.processing.org/ 


*Processing* er integreret udviklingsmiljø 
(engelsk: integrated development environment, IDE)
som gør det muligt at skrive programmer i 
programmeringssprogene Java og Python 2.7,
udføre/afvikle/køre dem.

Processing-miljøet gør det let at skrive programmer som har grafiske elementer. 
Processing er designet som et værktøj til at lære programmering
med en grafisk eller kunstnerisk tilgang.

.. warning:: Python Mode til Processing undersøtter kun Python 2.7.
    Bogen her antager du benytter Python 3, visse emner Python |py-ver|.

    til kapitel :ref:`sec-basic-programming`.
